## 개발환경

- back-end
  - java11
  - gradle
  - spring-boot 2.5.3
  - firebase

- front-end
  - HTML5
  - SASS
  - JavaScript(ES6)
  - Node.js
  - ReactJS

## Usage

### back-end

```sh
$ cd Back-End
$ ./gradlew clean build
```

### front-end

```sh
$ cd Front-End
$ npm run start
```
